Question 2 — Spinner & probability distribution
2(a) 1 mark
A spinner lands on 2, 4, 5, 7 or 8 only. The probability distribution of X is:
x24578
P(X = x)0.250.30.20.10.15
Find P(2X − 3 > 5)
Solve the inequality for X first: 2X − 3 > 5 → 2X > 8 → X > 4. Which values in the table satisfy this?
2X − 3 > 5  →  X > 4

Values of X that are greater than 4: 5, 7, 8

P(X > 4) = P(X=5) + P(X=7) + P(X=8)
= 0.2 + 0.1 + 0.15 = 0.45
2(b) 3 marks
Given that E(X) = 4.6, show that Var(X) = 4.14

✗ Show-that question — answer verification not applicable.

Use the formula Var(X) = E(X²) − [E(X)]². Calculate E(X²) = Σx²·P(X=x), then subtract 4.6².
E(X²) = 2²×0.25 + 4²×0.3 + 5²×0.2 + 7²×0.1 + 8²×0.15
= 1 + 4.8 + 5 + 4.9 + 9.6 = 25.3

Var(X) = E(X²) − [E(X)]²
= 25.3 − 4.6² = 25.3 − 21.16 = 4.14 ✓
2(c) 4 marks
The random variable Y = aX − b where a and b are positive constants.
Given E(Y) = 13.4 and Var(Y) = 66.24, find the values of a and b.
Use E(Y) = a·E(X) − b and Var(Y) = a²·Var(X). Substitute E(X) = 4.6 and Var(X) = 4.14 to form two equations.
From Var(Y):
66.24 = a² × 4.14  →  a² = 16  →  a = 4

From E(Y):
13.4 = 4 × 4.6 − b = 18.4 − b  →  b = 5
2(d) 4 marks
Sam and Alex each spin the spinner once, landing on X₁ and X₂ respectively.
Sam's score: S = X₁  |  Alex's score: R = 2X₂ − 3
The higher score wins; equal scores = draw.

Find the probability that Sam wins the game.
Sam wins when X₁ > 2X₂ − 3. First find what values Alex's score R can take:
R = 2×{2,4,5,7,8} − 3 = {1, 5, 7, 11, 13}
For each value of R, find which values of X₁ beat it and sum the probabilities.
Alex's possible scores R and when Sam wins (X₁ > R):

• R = 1 (X₂=2, p=0.25): any X₁ wins  → 0.25 × 1 = 0.25
• R = 5 (X₂=4, p=0.3): X₁ ∈ {7,8}  → 0.3 × (0.1+0.15) = 0.075
• R = 7 (X₂=5, p=0.2): X₁ = 8 only  → 0.2 × 0.15 = 0.03
• R = 11 or 13: X₁ cannot exceed these  → 0

P(Sam wins) = 0.25 + 0.075 + 0.03 = 0.355
Question 3 — Pebbles: histograms, median & coding
3(a) 3 marks
The lengths (mm) of 50 pebbles are summarised below:
LengthFrequency
20 ≤ x < 302
30 ≤ x < 3216
32 ≤ x < 3620
36 ≤ x < 408
40 ≤ x < 453
45 ≤ x < 501
A histogram is drawn. The bar for 32 ≤ x < 36 is 2.5 cm wide and 7.5 cm tall.
Calculate the width and height of the bar for 30 ≤ x < 32.
In a histogram, the area of each bar is proportional to frequency.
Bar area = width × height. Find the area per unit frequency using the known bar (32≤x<36), then use it for 30≤x<32.
Width: Class 30≤x<32 has width 2mm, same scale as 32≤x<36 (width 4mm → 2.5cm).
Scaling: 2.5 ÷ 4 × 2 = 1.25 cm

Height: Area of 32≤x<36 bar = 2.5 × 7.5 = 18.75 cm² (represents freq 20).
Area per frequency unit = 18.75 ÷ 20 = 0.9375 cm²
Area for 30≤x<32 (freq 16) = 16 × 0.9375 = 15 cm²
Height = 15 ÷ 1.25 = 12 cm
3(b) 2 marks
Using linear interpolation, estimate the median of x.
The median is the 25th value (n/2 = 25). Build a cumulative frequency table. Find which class contains the 25th value, then interpolate within that class.
Cumulative frequencies:
 up to 30: 2  |  up to 32: 18  |  up to 36: 38

The 25th value falls in 32 ≤ x < 36 (need 25−18 = 7 more values into this class).

Median = 32 + (7/20) × 4 = 32 + 1.4 = 33.4
3(c) 2 marks
The weight w grams of each pebble is coded using 10y = w − 20.
The coded data gives:   Σy = 104   and   Σy² = 233.54

Show that the mean of w is 40.8.

✗ Show-that question — answer verification not applicable.

Find ȳ = Σy/n. Then reverse the coding: since 10y = w − 20, we get w = 10y + 20. Apply this to the mean.
ȳ = 104 ÷ 50 = 2.08

Since w = 10y + 20:
w̄ = 10 × 2.08 + 20 = 20.8 + 20 = 40.8 ✓
3(d) 4 marks
Calculate the standard deviation of w. (Σy² = 233.54)
Find Var(y) = Σy²/n − ȳ² first.
Then use the coding: since w = 10y + 20, we have sd(w) = 10 × sd(y).
Var(y) = 233.54/50 − 2.08² = 4.6708 − 4.3264 = 0.3444
sd(y) = √0.3444 ≈ 0.5868

Since w = 10y + 20:   sd(w) = 10 × 0.5868 = 5.868 ≈ 5.87
3(e) 3 marks
A pebble weighing 40.8 g is added to the sample.
Without further calculation, state the effect on:
  (i) the mean of w
  (ii) the standard deviation of w

✗ Written answer required — answer verification not applicable.

The current mean is 40.8. If you add a value equal to the mean, what happens to the mean?
For standard deviation: does adding a value at the mean make the data more spread or less spread?
(i) Mean: Adding a value equal to the mean leaves the mean unchanged at 40.8.

(ii) Standard deviation: The new value sits exactly at the mean, pulling the distribution closer together — so the standard deviation decreases.

Both answers require a reason for full marks.
